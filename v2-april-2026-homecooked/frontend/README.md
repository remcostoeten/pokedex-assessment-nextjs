# Frontend

For :the back-end to work run `dotnet run` `../PokedexAPI/`. Alternative, for a work around if you can't get the back-end to work you can run a production build which uses a in browser set of the database instead of the in memory.

Go to the front-end dir and run `bun run build`, wait for that to complete and then `bun next start`. Or pnpm.

Furthermore see [ASSESSMENT.md](./ASSESSMENT.md).
